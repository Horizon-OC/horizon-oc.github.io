# Error Codes
*Most of this list comes from jimmy_f*

| Symptom | Likely Cause |
|---------|-------------|
| System/game hangs | GPU or RAM |
| Pressing the power button for 3–5 seconds brings up the Power Menu | GPU |
| Forcing shutdown causes the Switch to make a noise | RAM |
| Black screen with **"A fatal error occurred when running Atmosphere"** | Probably CPU, can be RAM too |
| Atmosphere blue screen with **"An error has occurred"** | Probably RAM, can be CPU too |
| System does not boot at all | CPU |
| Game crash with **"The software was closed because an error occurred"** | Can be anything |
| Graphical glitches (minor) | Most likely GPU |
| Graphical glitches (major) | Most likely RAM |
| **"Please connect a controller"** screen (controller was not disconnected) | CPU or RAM |
| <span style="display:inline-flex;align-items:center;gap:6px;"><span style="width:12px;height:12px;border-radius:50%;background:#FF7700;display:inline-block;"></span> Orange screen crash</span> | GPU instability or KIP corruption|
| <span style="display:inline-flex;align-items:center;gap:6px;"><span style="width:12px;height:12px;border-radius:50%;background:#FF0000;display:inline-block;"></span> Red screen on boot</span> | KIP Corruption |
| <span style="display:inline-flex;align-items:center;gap:6px;"><span style="width:12px;height:12px;border-radius:50%;background:#00FFFF;display:inline-block;"></span> Cyan screen on boot</span> | KIP Corruption |
| <span style="display:inline-flex;align-items:center;gap:6px;"><span style="width:12px;height:12px;border-radius:50%;background:#8800FF;display:inline-block;"></span> Purple screen on boot</span> | Patching fail or NAND Corruption. |


## Notes

- **Minor graphical glitches** generally indicate a GPU issue.
- **Major graphical glitches** generally indicate a RAM issue.
- Some symptoms can be caused by multiple components, so these are only indicators, not definitive diagnoses.
